# Empleo Privado en Argentina (2007 - 2022)

Proyecto de análisis de datos realizado en Python.

## Descripción

Análisis de la evolución de los puestos de trabajo asalariados registrados 
en Argentina entre los años 2007 y 2022, segmentado por sector de actividad económica.

## Dataset

Datos obtenidos del portal de datos abiertos del Ministerio de Producción de Argentina.

- Página principal: https://datos.produccion.gob.ar/dataset/puestos-de-trabajo-por-sector-productivo
- Dataset utilizado: https://datos.produccion.gob.ar/dataset/puestos-de-trabajo-por-sector-productivo/archivo/3effd219-24e9-4d8a-bb38-0636e11290b6
- Diccionario de CLAEs: https://datos.produccion.gob.ar/dataset/puestos-de-trabajo-por-sector-productivo/archivo/0c48706b-94e0-4ba9-abac-7a02a44cd840

Archivos incluidos en el repositorio:
- `puestos_privados_mensual_por_clae6.csv`: puestos de trabajo asalariados registrados por mes y actividad
- `clae_agg.csv`: diccionario de clasificación de actividades económicas (CLAEs)

Los dos datasets se integran mediante una clave en común.

## Tecnologías utilizadas

- Python
- NumPy
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## Contenido

- Evolución temporal de los puestos de trabajo registrados por año
- Segmentación por sector de actividad económica
- Visualizaciones para análisis comparativo entre períodos
