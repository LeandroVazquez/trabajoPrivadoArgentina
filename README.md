# Empleo Privado en Argentina (2007 - 2022)

Proyecto de análisis de datos realizado en Python.

## Descripción

Análisis de la evolución del empleo privado en Argentina entre los años 2007 y 2022, 
segmentado por categorías o sectores de actividad económica.

## Dataset

Datos obtenidos de fuentes públicas oficiales, disponibles en los archivos CSV 
incluidos en el repositorio:
- `puestos_privados_mensual_por_clae6.csv`: puestos de trabajo privados por mes y actividad
- `clae_agg.csv`: clasificación de actividades económicas

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Contenido

- Evolución temporal del empleo privado por año
- Segmentación por categorías o sectores de actividad
- Visualizaciones para análisis comparativo entre períodos
